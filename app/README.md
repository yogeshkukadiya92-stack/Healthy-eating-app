# Aura Diet Mobile

Run after installing Flutter stable:

```bash
flutter pub get
flutter run --dart-define=SUPABASE_URL=https://your-project.supabase.co --dart-define=SUPABASE_ANON_KEY=your-anon-key
```

The current mobile app includes the polished shell, demo providers, premium UI components, and dependency hooks for Supabase, camera, barcode scanning, biometrics, push notifications, local storage, charts, and RevenueCat.
