# Roadmap

## Phase 1

- Complete Supabase auth guard and onboarding persistence.
- Replace demo home data with realtime Supabase providers.
- Implement food search, custom food creation, recent foods, and favorites.
- Add offline queue processing.

## Phase 2

- Wire camera, image upload, barcode scanner, OCR, and AI food analysis.
- Add editable AI preview with correction tracking.
- Add water, weight, and goal screens with real mutations.
- Add notification scheduling.

## Phase 3

- Integrate RevenueCat subscriptions and entitlement gates.
- Add advanced analytics, PDF/XLSX/CSV exports, and weekly AI reports.
- Add HealthKit, Google Fit, Fitbit, and widgets.
- Harden admin panel with SSO and audit logs.

## Phase 4

- Production QA, accessibility audit, crash reporting, store metadata, privacy policies, and launch builds.
